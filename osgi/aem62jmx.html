  • The below table missing the bean details for special features.Example:- ldap bean (as it is available after enabling the ldap feature)
  • The below table provided complete attributes list however default value is listed only for attributes that can be configurable. Example:- BackupDelay
  • The table list attributes/operation for bean type rather than listing for all names. Ex:- Rather than repeating same attributes/operation for all replication agents like publish,offloading,flush etc..... it is just Shown once under respective type "DomainName = com.adobe.granite.replication type= agent"
  • More details refer https://docs.adobe.com/docs/en/aem/6-2/administer/operations/jmx-console.html
